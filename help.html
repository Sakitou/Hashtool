HashTool Help

Welcome to HashTool, a Python script for calculating and comparing hashes by Sakitou.

Usage:

        python hashtool.py [options]

        Options:
        -s, --string     Calculate hashes for the provided text.
        -f, --file       Calculate hashes for the contents of the file.
        -c, --compare    Compare two files.
        -i, --interface  Launch the GUI interface.
        -hh, --help_html Open this help page in the default web browser.

Examples:

        python hashtool.py -s "Hello, World!"
        python hashtool.py -f example.txt
        python hashtool.py -c file1.txt file2.txt
        python hashtool.py -i
        python hashtool.py -hh

Installation:

        sudo cp hashtool.py /usr/local/bin/htool && sudo chmod +x /usr/local/bin/htool
